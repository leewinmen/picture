受益人
2019-12-25 / 人性 喜剧 / ★★★★☆ 推荐
讲了一段稀里糊涂的故事，塞了一些观众爱看的东西，其实可以看到很多其他电影的影子，也能很明显地感受到柳岩和大鹏对于转型的急迫需求，不管怎么说，能够卸妆说出：“我今年三十八岁了”的柳岩，已经和从前大不相同了，
疯狂的外星人
2019-12-25 / 春节档 贺岁 / ★★★☆☆ 还行
能是今年春节档里唯一的作者电影。因为它太不爆笑了。全片贯穿一个外星人＞美国人＞中国人＞猴的生物鄙视链，却被链条金字塔尖的“最高科技”逆转，变成了一个哲学上“往前迈了一步”的退化论。最后齐天大圣的脑洞真的是多重语境的混杂，你可以看成是对这些年来春节档的鄙视，也可以看成是一次神话对科学的鄙视
飞驰人生
2019-12-25 / 喜剧 搞笑 / ★★★☆☆ 还行
飞驰的不是人生，是日益飙红的票房，和持续降低的追求。韩寒还是那个韩寒，抖完段子撒热血，前半段是小品拼凑，各自割裂的廉价笑点，后半段是漫画合集，硬凹出来的励志姿态。剪掉的部分换来了更多排片，飞向夕阳后一句“献给XX”，何其容易，流行歌曲、慢镜头、低劣特效，屡试不爽的三板斧。
孟买酒店 / Hotel Mumbai
2019-12-25 / 恐怖袭击 印度 / ★★★★★ 力荐
100分钟近乎窒息的观影体验，不到20分钟就进入剑拔弩张的紧张氛围并且一直延续到了最后，要是影像上更写实一点就真的是一部极其出色的真实事件改编惊悚片了，不过就目前导演的水平来说，依然让影片高潮迭起、刺激抓人，群戏的处理更是非常到位，恰到好处。
绿皮书 / Green Book
2019-12-25 / 黑人平权 种族 / ★★★★★ 力荐
基层白人司机护送著名黑人钢琴家在美国南部巡回演出的故事。两位角色的社会阶级和教育程度的差异，让电影笑料百出，肤色的对比却又十分的讽刺，让电影充满种族平权的抗争和泪点。两位演员默契火花十足，是一个十分温暖人心的故事，影片结束之后响起经久不息的掌声，边笑着边流着眼泪鼓掌到手都疼了。
寄生虫 / 기생충
2019-12-25 / 韩国 人性 / ★★★★★ 力荐
沐浴庭院里的阳光就能进化成人，呼吸暗室里的空气只会走肉如鬼。抱上贫穷的石头终生无法脱手，染上廉价的气味永世不能消除。蜷缩地下的臭虫，用药剂开窗替自己杀菌；位居高处的宿主，用金钱雇人给自己消毒。富人吸着穷人的血高升，穷人寄在富人的屋苟活，人鬼同处一室分享食物，虫兽寄生一体发育恩仇。
调音师 / Andhadhun
2019-12-25 / 印度 反转 悬疑 / ★★★★★ 力荐
2018年看过，剧本，配乐，运镜无一不精，数次反转也做相当流畅，好几处玩梗更是让人笑到喷饭。好久没体会到带着智商看电影的快乐了，向印度电影人致敬！
流浪地球
2019-12-25 / 科幻 冒险 灾难 / ★★★★★ 力荐
华语真正意义上的第一部科幻大片！刘慈欣的硬核科幻设定，小兵扛大旗的主流价值观人设，东方式的家庭理念核心，地心引力与星际穿越的融合。故事铺排+情节张力+情绪渲染+镜头调度非常娴熟，意料之外的特效大场面超赞。
少年的你
2019-12-25 / 校园暴力 青春 / ★★★★★ 力荐
不是每个陈念都能遇上愿意用所有未来去保护她的小北，但希望每个陈念都能像雨后的小雏菊一样，在角落里坚强绽放。那些打不倒我们的使我们更坚强。
